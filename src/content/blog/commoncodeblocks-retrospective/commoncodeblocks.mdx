## kotlin 
### 添加全局logger
```
val <T : Any> T.LOGGER: Logger
    get() = LoggerFactory.getLogger(this::class.java)
```

## linux
### 打印文件树
```
find . -maxdepth 2 | sed -e 's;[^/]*/;|____;g;s;____|; |;g'
```
